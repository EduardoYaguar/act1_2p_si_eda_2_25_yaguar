# Hospital Readmission Prediction After a Discharge [Diabetic Patients]

The purpose of this project is to analyze hospital readmission patterns considering diabetic patients using data from 130 U.S. hospitals for the years 1999 - 2008.
The main objective is to build a model that predicts whether a patient will be readmitted to the hospital after discharge or not.

## Dataset Information

**Source:** https://archive.ics.uci.edu/dataset/296/diabetes+130-us+hospitals+for+years+1999-2008

### Dataset Characteristics:
- **Total Records:** 101,766 encounters
- **Features:** 50 variables
- **Time Period:** 1999-2008
- **Hospitals:** 130 US hospitals
- **Patient Type:** Diabetic patients
